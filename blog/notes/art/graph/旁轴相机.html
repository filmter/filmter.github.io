胶片相机 #

旁轴 Rangefinder Camera #

旁轴取景式相机，由于取景光轴位于镜头成像光轴旁边，与之平行，因而取名 旁轴相机。不同于单单反拍摄的所见即所得，旁轴相机的实际拍摄的画面和观察的画面，会有偏差。

  • 旁轴没有反光镜，这样相机较薄，机震较轻。
  • 旁轴相机由于对焦基线的原因和没有反光镜的限制，一般广角镜头可以做得非常贴近底片，结构和成像也比单反的要好。所以广角和标准焦距是它的强项，旁轴最长焦极限是 135mm。

选购建议 #

  • 手动快门(非联动测光)：要买快门可手动调节的相机。因为由于年代久远，自动测光模块很大可能会坏掉，如果测光模块坏掉而机器又不能手动调整快门速度，这台机器就不能够进行准确的曝光。并且测光模块是需要用到电池的，老款相机的电池很大可能已经停产。
  • 黄斑对焦：如果机器没有对焦功能可以购买测距仪装在热靴口上。

机器测试 #

  • 镜头：

    • 看镜头外围的磨损情况，滤光镜接口有无变形（磕陷），橡胶圈有无脱离或松动，对焦环转动是否顺滑（有些老镜头由于润滑剂干涸，转动困难）
    • 光圈环转动时挡位是否清晰，叶片开合是否正常，叶片上有没有油渍（漏油），如果是变焦镜头，看看变焦环转动是否自如。
    • 镜头底部的螺丝有没有滑丝的痕迹（如有，说明镜头被拆卸过）。
    • 如果镜头是改过口的，看看是原厂改口还是私自改口，私自改口的水平如何。
    • 前后玻璃是否干净，有无划痕。
    • 把光圈开到最大，逆光看玻璃是否通透，有无灰尘颗粒，霉斑（霉斑是很难清理的）。
    • 把镜头装上相机检查镜头光圈的开合情况：把快门速度设在最慢速度或B门，在不同的光圈下按快门，看光圈叶片的开合大小是否与光圈数值相符。
    • 最后，如果是收藏级的镜头，还要核对镜头序列号，以免买到非原装的镜头。

  • 过片/回片：

    • 对于手动上卷的相机，掰动过卷杆，每次过卷之后能否按快门？胶卷张数计数器显示是否递增？
    • 打开后盖，看过卷轴是否正常转动，再看看倒片轴是否转动自如。当然最好是拿一个废卷来试一下。
    • 对于自动上卷的相机，一定要用一个胶卷来试验是否能有效上卷、过卷、和倒卷。遇到过一个自动相机，没装胶卷时可以正常测光和对焦，好像没有问题，但就是不能自动上胶卷，手动装了胶卷也不能自动过卷，等于一个废机。

  • 快门：

    • 拉动过片杆，按快门，听快门的声音，没有仪器虽然不能精确测定快门速度，但可以大体上听出来，不同速度的时间间隙。
    • 在后盖打开的情况下，不同的快门速度不仅可以听出来，还可以看到快门帘开启的时间间隔。
    • 每次按快门时反光镜是否正常起落？每次过卷时手感是否顺畅？有无力度不均的感觉？力度不均让我担心将来某天会卡住。
    • 测试胶片相机快门是否良好的时候，一般测试它的慢门（比如：1s），一般慢门如果可以正常准确的工作的话，这台机器的快门应该是完好的

  • 快门帘：

    • 遮光布帘易发霉，看遮光布帘是否霉变
    • 快门帘分横走布帘和纵走金属帘：
      • 打开相机后盖，看布帘是否平整，有无漏洞，拉一下过片杆，布帘会走过，看新的一段布帘是否完好。
      • 金属帘是否有缝隙（漏光的隐患），同样要拉一下过片杆，看另一段金属帘是否完好。

  • 后盖：

    • 密封海绵老化，造成漏光的隐患，我倒不太在意，因为很容易更换，不是大问题，就像你的鞋袜不能穿一辈子一样。
    • 但后盖能否正常开关是大问题，后盖卡扣有无损坏迹象？若没问题，可以关上后盖了。

  • 相机底部：

    • 三脚架接口螺纹有无损坏
    • 电池盖能否正常打开（不少相机的电池盖被拧坏了，很难打开）
    • 电池仓内有无锈迹
    • 倒卷按钮是否正常
    • 卷片马达接口有无损坏

  • 取景器检查：

    • 看取景器视窗，首先要看视窗内是否干净，许多相机的对焦屏是不能拆卸的，如果不干净，或有划痕，尤其是霉斑，很难解决。
    • 然后看取景器信息显示是否正常：光圈显示、速度显示，指针或LED曝光显示，LED液晶是否有漏液现象，不同的机器有不同的显示。如果电池没有问题，而显示不正常（比如指针不动），那就是相机有问题。
    • 在不同的光圈数值下，按景深键，看景深变化是否正常

  • 测光：

    • 先保证电池是好的，否则无法测光。
    • 打开电源，把镜头对准明亮物体，看取景器内曝光显示，是 + 还是 - ，调整镜头光圈，从大到小，看 + / - 是否正常变化（光圈缩小时，曝光应该从 + 变到 - ，反之亦反）。
    • 如果有一台数码相机，可以比较一下数码相机的测光值和胶片相机的测光值。也可以下载一个手机测光app，将手机的测光值与相机的测光值比较，看是否一致。
    • 然后固定光圈，调节快门速度，看 + / - 是否正常变化（速度从慢到快时，曝光应该从 + 变到 - ，反之亦反）；
    • 然后在光圈和速度都固定的情况下，调节感光度ASA，看 + / - 是否正常变化（ASA由大变小时，曝光应该从 + 变到 - ，反之亦反）；
    • 然后在以上参数都固定时，调节曝光补偿，看 + / - 是否正常变化（曝光补偿由大变小时，曝光应该从 + 变到 - ，反之亦反）。
    • 完成以上步骤后，再把镜头对准较暗的物体，重复以上操作，看 + / - 是否正常变化。
    • 这一套做下来，光圈，快门速度，测光元件，ASA调节，和曝光补偿等功能都检查到了。
    • 对于自动相机，还要检查 M, A, S, P 等不同曝光模式，以及点测光，矩阵测光等，这取决于具体的相机。

  • 对焦：

    • 黄斑看黄斑是否清晰，横纵方向是否都能合焦
    • 如果是手动对焦相机，检查对焦环是否转动自如，视窗内看到的聚焦目标的距离与镜头上的距离刻度是否大致相同。裂像式或微菱式对焦是否清晰。
    • 如果是自动对焦相机，在不同光线条件下，选择不同距离的目标聚焦，看聚焦是否准确，快速，有无异常噪音。

  • 自拍：

    • 老相机自拍功能慎用，有造成机械内部卡住的风险
    • 测试自拍计时是否有效，可以准确释放快门

  • 闪光灯

    • 如果有内置闪光灯，看闪光灯能否正常抬起，正常打闪。

  • 多重曝光：

    • 如果有多重曝光功能，多重曝光后，看看胶卷张数是否变化。

  • 感光度：

    • 是否可调节（宾得SP提起来旋转）

维修保养 #

  • 取景器黑线：

    • 很可能是五棱镜镀银脱落造成的，没有解决方案。

  • 反光镜不下落的原因分析：

    • 门帘卡住
    • 卷片上档机构里面有空气阻尼器，阻尼器里面的橡胶垫老化粘住，空气阻尼器一般在相机底部，需要将反光板系统拆开。
    • 润滑油老化变粘了，需要打理
    • 机子长期不用机械和弹簧不利落造成的

  • 过片扳手掰不动，大概有两种可能：

    • 过片扳手卡住，导致不能过片，可以拆开底盖和上面看看有没有不在位置上的零件。
    • 快门卡住，过完一次片后快门不能激发或者卡死，也会导致过片扳手掰不过来。

  • 测光不动

    • 可能里面卡住了
    • 也有可能电路腐蚀断了

宾得SP #

B门不回落问题分析 #

首先看下 宾得sp 的底部构造： 快门按下后的正常流程：齿轮1上面有小凸块（如果看不到过片后就会转出来啦），当快门按下后齿轮1旋转，凸块撞击3使4回落。

故障在于齿轮1在B门的时候卡在了起始点并没有转动，因此导致4无法回落。

分析原因：

  • 两个齿轮轴不润滑
  • 幕帘轴不润滑
  • 张力弹簧力量减弱 以上都有可能。这些部位都应该是干性润滑，有的被加了不合适的润滑油也会这样造成无法顺畅旋转。

PS：选购润滑油也需要考虑润滑油的适宜温度，例如不耐低温的润滑油，很可能导致冬天无法正常使用。

如何测量电池的电量 #

快门调至B门，ISO调整到100，推动测光按钮，如果针头快速下降，表示电量充足。

快门速度拨盘左边的小三角，变成红色是什么意思 #

三角指针变成红色的时候，表示 快门速度和感光度的设置 已经超过测光表的测量范围
